# 🏙️ Smart City Grievance Portal

## 📌 Project Overview

The Smart City Grievance Portal is a web-based complaint management system developed using Flask, Python, HTML, CSS, and SQLite. It enables citizens to submit grievances while allowing administrators to classify, prioritize, monitor, and manage complaints through an interactive dashboard.

---

## ✨ Features

- Complaint Submission Portal
- Automatic Complaint Categorization
- Priority Detection
- Anomaly Detection
- Complaint Status Tracking
- Search and Filter Complaints
- Dashboard Statistics
- Responsive User Interface
- SQLite Database Integration

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/Haritha457/Smart-City-Grievance-Portal.git
```

Move into the project directory

```bash
cd Smart-City-Grievance-Portal
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
