# 🏙️ Smart City Grievance Portal

An AI-powered grievance management platform designed for smart cities, gated communities, campuses, and enterprise facilities.

The platform enables citizens to submit complaints, track their progress using unique tracking IDs, and allows administrators to efficiently manage and resolve issues through an analytics dashboard.

## ✨ Features

- 📝 Complaint Submission Portal
- 🤖 AI-Based Complaint Categorization
- ⚡ Automatic Priority Assignment

- 📊 Interactive Admin Dashboard
- 🔎 Search and Filter Functionality
- 📈 Real-Time Analytics and Statistics

## ⚙️ System Workflow

Citizen submits complaint
        ↓
AI categorizes complaint
        ↓
Priority assigned automatically
        ↓
Tracking ID generated
        ↓
Admin reviews complaint
        ↓
Status updated
        ↓
Citizen tracks complaint progress

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/Haritha457/Smart-City-Grievance-Portal.git
```

Move into the project directory:

```bash
cd Smart-City-Grievance-Portal
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
